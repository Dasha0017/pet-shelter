📚 Документация API Pet Shelter

Базовый URL: http://localhost:{{.Port}}/api

GET /api/animals

Получить список всех животных

curl http://localhost:{{.Port}}/api/animals
POST /api/admin/animals

Добавить новое животное (требует заголовок X-Admin: true)

curl -X POST http://localhost:{{.Port}}/api/admin/animals \
 -H "Content-Type: application/json" \
 -H "X-Admin: true" \
 -d '{"name":"Мурзик","species":"Кошка","age":1}'
GET /api/catalog

Получить каталог товаров

curl http://localhost:{{.Port}}/api/catalog
POST /api/login

Войти в систему (admin/admin123)

curl -X POST http://localhost:{{.Port}}/api/login \
 -H "Content-Type: application/json" \
 -d '{"username":"admin","password":"admin123"}'
