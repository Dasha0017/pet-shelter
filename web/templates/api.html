📚 Документация API

REST API для работы с системой Pet Shelter

Базовый URL: http://localhost:{{.Port}}/api

Формат данных: JSON

Аутентификация: Заголовок X-Admin: true для админских endpoints

📡 Доступные endpoints

GET /api/animals

Получить список всех животных в приюте

POST /api/admin/animals

Добавить новое животное (требует заголовок X-Admin: true)

GET /api/catalog

Получить каталог товаров для животных

POST /api/login

Войти в систему (admin/admin123)
